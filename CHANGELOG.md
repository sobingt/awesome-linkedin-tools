## How to Report Changes

To report a change or new tool for the next release:

1. Open an issue with label `tool-suggestion` or `enhancement`
2. Submit a PR with changes
3. Changes will be reviewed and added to the CHANGELOG

## Versioning

- **Major (X.0.0)**: Significant restructure (e.g., new categories, major redesign)
- **Minor (0.X.0)**: New tools, features, or sections added
- **Patch (0.0.X)**: Bug fixes, corrections, link updates
