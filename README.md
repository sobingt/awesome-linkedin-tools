## 7) FAQ

**Q: Are these tools safe to use?**  
A: LinkedIn actively restricts automation. Use tools conservatively, respect rate limits, and always review the latest LinkedIn Terms of Service. High-risk tools include automation frameworks and scrapers.

**Q: What's the difference between extensions and SaaS tools?**  
A: Extensions run in your browser and are often cheaper but less scalable. SaaS tools run on remote servers and offer better multi-account management and compliance features.

**Q: Which tools are best for compliance?**  
A: Enterprise CRM tools like HubSpot and Salesforce have built-in audit trails and compliance features. Avoid raw scraping tools if compliance is critical.

**Q: Can I combine multiple tools?**  
A: Yes! Use automation frameworks (Zapier, Make) to connect tools. Common stacks: Outreach tool + Email enrichment + CRM.

**Q: What's the ROI on these tools?**  
A: Varies widely. A recruiter might see 3-6 month payback on a $50/mo tool. Larger teams investing in CRM integration may take 6-12 months to see ROI through pipeline visibility.

## 9) How to contribute

1. Fork this repository.
2. Add tools under the appropriate category using the same table format.
3. Include a one-line description, homepage link, and tags from the legend.
4. Open a pull request with clear notes and label the PR (e.g., `automation`, `enrichment`).

**Submission checklist**

* Tool name and canonical homepage
* One-line description (max 20 words)
* Tags from the legend (at least one)
* Category (Outreach / Task Automation / Automation frameworks / Data Extraction / Multichannel)
* Risk level: `low`, `medium`, or `high` (automation/scraping risk)
