## How to Use This Matrix

1. **Identify your needs** - What features are must-haves?
2. **Filter by tool type** - Outreach? Enrichment? Automation?
3. **Check risk levels** - Low risk = safer, High risk = higher ban potential
4. **Compare pricing** - Note if costs scale with seats/accounts
5. **Test before buying** - Most tools offer free trials

## Important Notes

- **Pricing is approximate** - Confirm current pricing on tool websites
- **Features change frequently** - Last updated Jan 2026, check tool sites for latest
- **API availability varies** - Some tools restrict API access by tier
- **Risk levels reflect LinkedIn policy** - Automation may violate TOS
- **Multi-account support** - Some require separate subscriptions per account

## Quick Recommendations by Use Case

### "I have $100/month and need outreach"
→ **Dripify**, **Waalaxy**, or **Dux-Soup**

### "I need emails for a list of 10,000 prospects"
→ **Apollo.io**, **ZoomInfo**, or **Clearbit API**

### "I want to build a custom automation"
→ **Playwright**, **Puppeteer**, or **Selenium**

### "I need everything: CRM + Email + LinkedIn"
→ **HubSpot** (mid-market) or **Reply.io** (SMB)

### "I'm a solo recruiter with a tight budget"
→ **Dux-Soup** (extension) + **Hunter** (email finder)

### "I need compliance & audit trails"
→ **HubSpot**, **Salesforce**, or **Reply.io**
